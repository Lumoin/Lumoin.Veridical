The harness compiles a small GF2_128<4> circuit through the real compiler. The full field is 16-byte
GF(2^128) (matching the existing C.2-C.6 Ligero stack); the subfield is GF(2^16). The circuit is the
field-satisfiable relation w == (x + y) * (x + z) * x (x public, y/z/w private), chosen so logc = 0 (no
copies; the reference's write_sc_proof requires logc == 0) and nl = 3 (a genuine multi-layer walk with
per-layer logw of 2, 3, 3). The fixed witnesses are x = of_scalar(3), y = of_scalar(5), z = of_scalar(7),
w = the field product; W[0] is the constant-one wire. The deterministic Fiat-Shamir seed is "sc7" (ASCII).

The sc byte layout, per layer (write_sc_proof, gated on logc == 0):
  for wi in 0..logw:                  // one binding round per hand variable
    for k in {0, 2}:                  // the k != 1 optimization: p(1) is implied and not sent
      write hp[0][wi].t_[k]           // left hand, then
      write hp[1][wi].t_[k]           // right hand   (k outer, hand inner)
  write wc[0]                         // the two claims on W[L,C], W[R,C] for the next layer
  write wc[1]
Each element is 16 little-endian to_bytes_field bytes. Per-layer byte size = (logw*(3-1)*2 + 2)*16; the
total here is (2*4+2 + 3*4+2 + 3*4+2)*16 = (10 + 14 + 14)*16 = 608 bytes. read_sc_proof derives the same
needed-bytes formula (logw*(3-1)*2 + 2)*kBytes per layer and rejects on underflow; the unsent p(1) slots
read back as F.zero().

     absorbed [tag 2, count 2] as the next layer's two claims.

The GF(2^128) round-polynomial arithmetic: WPoly is degree-3 (N = 3), evaluated at points {0, 1, g} where
g is the subfield generator (poly_evaluation_point(0) = 0, point(i) = g^(i-1)); eval_lagrange converts the
Lagrange values to Newton forward differences (newton_of_lagrange) and evaluates (eval_newton) using the
field's newton_denominator(k, i) = (X[k] - X[k-i])^(-1).

the full ZK verifier composition (ZkCommon::verifier_constraints building the Ligero A/b system, then
LigeroVerifier) are above this segment and out of scope for C.7's sc-segment gate.
